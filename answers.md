# Answers

## What is an Org Development Model?

An Org Development Model defines how Salesforce environments are used throughout development, testing, and deployment.

---

## What is Salesforce DevOps?

Salesforce DevOps is the practice of managing development, testing, release, and deployment processes using automation and collaboration tools.

---

## What is a Deployment Workflow?

A deployment workflow is the process of moving Salesforce changes from development environments to production.

---

## What is Agentforce DX?

Agentforce DX is a Salesforce development experience used for building and managing AI-powered agents.

---

## What is Visualforce?

Visualforce is a framework for creating custom Salesforce user interfaces using tags similar to HTML.

---

## What is Aura?

Aura is a component-based framework used for building dynamic applications on the Salesforce platform.

---

## Difference Between Visualforce and Aura?

| Visualforce | Aura |
|------------|------|
| Page-based framework | Component-based framework |
| Server-side rendering | Client-side rendering |
| Older technology | Modern framework before LWC |

---

## Why is DevOps Important?

DevOps improves collaboration, deployment speed, quality, and release management.
